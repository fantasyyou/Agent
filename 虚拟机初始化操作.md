# 虚拟机初始化操作

## 系统更新

```bash
sudo apt update && sudo apt upgrade -y
```

## 安装 VMware Tools（共享桌面/剪贴板）

```bash
sudo apt install open-vm-tools open-vm-tools-desktop -y
```

## 安装 SSH 服务（远程连接）

```bash
sudo apt install openssh-server -y
sudo systemctl start ssh
sudo systemctl enable ssh
```

### 获取虚拟机 IP

```bash
hostname -I
# 或
ip addr | grep inet
```

### Windows 连接

```powershell
ssh musong@虚拟机IP
```

## 安装 Docker

```bash
sudo apt install docker.io -y
sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker $USER
```

## 启动 Elasticsearch

```bash
docker volume create es_data
docker run -d \
  --name es \
  --memory=4g \
  --memory-reservation=3g \
  -p 9200:9200 \
  -p 9300:9300 \
  -e discovery.type=single-node \
  -e ES_JAVA_OPTS="-Xms2g -Xmx2g" \
  -v es_data:/usr/share/elasticsearch/data \
  docker.elastic.co/elasticsearch/elasticsearch:8.11.0
```

## 安装 Python 环境

```bash
sudo apt install python3 python3-pip -y
pip install langchain chromadb sentence-transformers python-docx elasticsearch langchain-elasticsearch
```

## 配置共享文件夹

### VMware 设置

1. 虚拟机设置 → 选项 → 共享文件夹
2. 选择"总是启用"
3. 添加共享文件夹

### 挂载共享文件夹

```bash
mkdir ~/shared
sudo mount -t fuse.vmhgfs-fuse .host:/shared_folder ~/shared -o allow_other
```

### 开机自动挂载

```bash
sudo nano /etc/fstab
# 添加：
# .host:/shared_folder /home/musong/shared fuse.vmhgfs-fuse allow_other 0 0
```
